## Getting Started

to run project locally:

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

to run db locally:

```bash
touch .env
```

download `npx` and `prisma (google 'yarn add prisma')`
put db creds to the `env` file locally

```bash
npx prisma help
```
